To install Spring with Eclipse, first install Spring suite as outline here:
Use Help->eclipse marketplace  and search for spring suite and install it


Also need to install apache commons logging also outlined in this page
Then get the actual binaries by going to:
http://repo.spring.io/release/org/springframework/spring
and looking for latest file similar to name spring-framework-4.1.6.RELEASE-dist.zip

Use this website as reference point:
http://www.tutorialspoint.com/spring/spring_environment_setup.htm
